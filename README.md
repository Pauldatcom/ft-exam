# Simple Node.js API

A simple Node.js API with Express that returns "hello world".

## Installation

```bash
npm install
```

## Running the API

### Development mode (with auto-restart)
```bash
npm run dev
```

## Usage

The API will be available at `http://localhost:3000`

### Endpoints

- `GET /` - Returns "hello world" message

Example response:
```json
{
  "message": "hello world"
}
```
